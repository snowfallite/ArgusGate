- **Клиентское приложение** — потребитель Gateway-API.
- **Шлюз (ArgusGate)** — reverse-proxy с 7-слойным конвейером детекции.

## Условные обозначения

- `●` — начало / конец процесса
- `◇` — решение (ветвление)
- `━━━` — fork / join: после `[DONE]` основной поток уже закрыт, фоновые
  задачи запускаются параллельно через `asyncio.create_task`
- `►` / `◄` — передача сообщения между участниками

## Почему часть терминов оставлена на английском

| Термин | Почему не переведён |
|---|---|
| **chunk** | Стандарт HTTP/SSE, в коде и логах везде так |
| **[DONE]**, **SSE** | Маркер конца стрима OpenAI-протокола |
| **jailbreak**, **prompt injection** | Устойчивые термины ИБ без однозначных аналогов |
| **PII** | Personally Identifiable Information — стандартное сокращение |
| **request_logs / detection_events** | Имена таблиц БД |
| **Presidio** | Название библиотеки Microsoft |
| **finalize** | Подписан как «дочистка» в самой диаграмме |
